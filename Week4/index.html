Week 4: Microcontroller Programming


Photoresistor controlling the On/Off of a LED IS THIS UPDATING


step 1: materials

  1. an LED
  2. photoresister
  3. (2) 300 ohm resisters
  4. jumper cables
  5. breadboard
  6. arduino UNO + usb cable

step 2: circuit

  • ground to ground rail on breadboard
  • 5v to power rail on breadboard
  • Pin 3 > 330ohm R > LED (+)
  • > LED (-) GND
  • 5V > photoresister
  • Analog 0 > 330ohm > photoresister

step 3: code


Lastly, plug your arduino into your laptop and set up the correct board and port. Then upload this code onto your board.

Here is the code that you should be able to copy and paste into your own IDE and configure to your own uses. I've commented the code to help make things more clear.



      int ledPin = 3;  //define the pin for the led
      int sensorInput = 0; //define the analog input for the photoresister

      void setup()  {
        pinMode(ledPin, OUTPUT);
      }



      void loop()  {

        sensorInput = (analogRead(0)/4); // divides input 0-1023 to resemble to 0-255

        analogWrite(ledPin, sensorInput);
        delay(20);
      }
